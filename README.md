# 🎵 OMusic

OMusic is a sleek and responsive music discovery web application that lets users search for virtually any song and instantly listen to a **30-second preview**. Built with modern web technologies, it provides a fast and intuitive way to explore music without requiring downloads or subscriptions.

## ✨ Features

* 🔍 Search for songs by title or artist
* 🎧 Instantly play a 30-second audio preview
* 🖼️ Displays album artwork and song details
* 📱 Fully responsive design for desktop and mobile
* ⚡ Fast and lightweight user experience
* 🎨 Clean, modern, and intuitive interface

## 🎯 How It Works

1. Enter the name of a song or artist.
2. Browse the search results.
3. Click on a track to listen to its available 30-second preview.
4. Explore new music quickly without leaving the page.
